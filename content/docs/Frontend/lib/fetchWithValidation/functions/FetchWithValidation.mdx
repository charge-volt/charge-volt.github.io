# Function: FetchWithValidation()

> **FetchWithValidation**\<`T`\>(`schema`, `url`): `Promise`\<`z.SafeParseReturnType`\<`T`, `T`\>\>

Fetches data from a specified URL and validates it against a provided Zod schema.

## Type parameters

• **T**

The type of data to be validated.

## Parameters

• **schema**: `ZodType`\<`T`, `ZodTypeDef`, `T`\>

The Zod schema used to validate the fetched data.

• **url**: `string`

The URL to fetch data from.

## Returns

`Promise`\<`z.SafeParseReturnType`\<`T`, `T`\>\>

A promise that resolves to a Zod safe parse result containing either the validated data or validation errors.

## Remarks

If the fetch request fails (response.ok is false), the function will return an error object
with the message from the backend response. The backend should ensure that error messages
don't contain sensitive information.
