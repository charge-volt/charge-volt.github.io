# Function: Render()

> **Render**(`params`): `Element`

Renders a file entry for a list of documents.

This component displays a file icon based on the file type and the file name.

## Parameters

• **params**

The parameters for rendering the file entry

• **params.data**

Data object containing file information

• **params.data.fileType**: `string`

• **params.value**: `string`

The display value (typically the file name)

## Returns

`Element`

A React component showing the file icon and name
